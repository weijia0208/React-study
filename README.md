# 2016级项目实训成果展示 

## 《蚂蚁寻物》 - HTML5与移动互联网应用开发

### 项目简介

蚂蚁寻物是一款包含基于地点闲置物品买卖的小程序。 旨在根据商品地点，方便广大师生对物品的选择。为各大高校的师生提供详细的分类二手物品，可以通过电话联系，线下进行交易。保障商品的可靠性，减少师生买卖物品地点上的局限性。
